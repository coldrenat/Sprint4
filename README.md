# Автоматизированные тесты для сервиса "Самокат"

## Описание проекта
Проект содержит автоматизированные тесты для сервиса аренды самокатов, написанные на Java с использованием Selenium WebDriver и JUnit 4.

## Технологии
- Java 11
- Maven
- Selenium WebDriver 4.14.1
- JUnit 4.13.2
- WebDriver Manager 5.6.2
- Page Object Pattern

## Структура проекта
src/test/java/
├── pages/ # Page Object классы
│ ├── MainPage.java
│ └── OrderPage.java
├── tests/ # Тестовые классы
│ ├── FAQTest.java
│ ├── OrderTest.java
│ └── LogoTest.java
└── BaseTest.java # Базовый класс для тестов


## Тестовые сценарии

### 1. Тестирование FAQ (Вопросы о важном)
- Проверка открытия/закрытия ответов на вопросы
- Параметризированный тест для всех 8 вопросов

### 2. Оформление заказа самоката
- Позитивный сценарий с двумя наборами данных
- Две точки входа: верхняя и нижняя кнопки "Заказать"
- Проверка успешного создания заказа

### 3. Дополнительные тесты
- Проверка логотипов (Самокат и Яндекс)
- Проверка поиска заказа по неверному номеру

## Запуск тестов

### Требования
- Java 11 или выше
- Maven 3.6+
- Google Chrome

### Команды для запуска
```bash
# Сборка проекта
mvn clean compile

# Запуск всех тестов
mvn test

# Запуск конкретного тестового класса
mvn test -Dtest=OrderTest

# Запуск с отчетом
mvn surefire-report:report
